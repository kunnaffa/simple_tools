# Language Switcher

Language Switcher is a small Python application that provides a graphical user interface (GUI) for switching between different keyboard languages on FreeBSD using the `setxkbmap` command.

## Features

- Allows switching between Hebrew, English, and Arabic keyboard languages.
- Simple and intuitive GUI with buttons for language selection.
- Uses the `setxkbmap` command to change the keyboard language.

## Requirements

- Python 3.x
- Tkinter library (usually included with Python installation on FreeBSD)

## Usage

1. Clone or download the repository to your FreeBSD machine.
2. Make sure you have Python 3.x installed.
3. Open a terminal and navigate to the project directory.
4. Run the following command to start the application: python ./lan.py
